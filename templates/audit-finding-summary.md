# Audit Finding Summary Template

## 401(k) Plan Audit Findings Documentation

### Purpose

This template provides a standardized format for documenting and communicating audit findings identified during a limited scope 401(k) plan audit. It includes:
- Finding documentation worksheet
- Management letter format
- Finding tracking log
- Corrective action plan template

---

[Describe what was found]

[Describe actual or potential impact]

Management Response:
[Include management's response and corrective action plan]


FINDING 2: [TITLE]
-------------------

[Continue format for each finding...]

PRIOR FINDING 1: [TITLE]

2. [Observation/recommendation]

## Common 401(k) Audit Findings Quick Reference

### Contribution Findings

| Finding | Typical Classification | Correction |
|---------|----------------------|------------|
| Late deposit of deferrals | SD or CD | VFCP + lost earnings |
| Incorrect deferral calculation | CD | Correct and make whole |
| Missed employer contribution | CD to SD | Contribute with earnings |
| Excess contributions | CD | Distribute excess + earnings |
| 415 limit exceeded | SD | Correct under EPCRS |

### Distribution Findings

| Finding | Typical Classification | Correction |
|---------|----------------------|------------|
| Incorrect vesting | CD to SD | Recalculate, make whole |
| Missing spousal consent | SD | Correct under EPCRS |
| Incorrect hardship | CD to SD | EPCRS correction |
| Missed RMD | CD | Distribute + file 5329 |
| Improper in-service | CD | EPCRS correction |

### Participant Data Findings

| Finding | Typical Classification | Correction |
|---------|----------------------|------------|
| Eligibility errors | CD to SD | QNEC if excluded |
| Incorrect compensation | CD | Recalculate contributions |
| Inaccurate census data | CD | Correct records |
| Missing beneficiary forms | CD | Obtain forms |

### Loan Findings

| Finding | Typical Classification | Correction |
|---------|----------------------|------------|
| Exceeds 50%/50K limit | SD | Deemed distribution |
| Improper term | CD to SD | EPCRS correction |
| Default not processed | CD | Process deemed distribution |
| Missing documentation | CD | Obtain documentation |

### Internal Control Findings

| Finding | Typical Classification | Correction |
|---------|----------------------|------------|
| Inadequate segregation | SD to MW | Implement compensating controls |
| No reconciliations | SD | Implement monthly reconciliation |
| Missing approvals | CD to SD | Implement approval process |
| No committee meetings | CD | Document meetings |
