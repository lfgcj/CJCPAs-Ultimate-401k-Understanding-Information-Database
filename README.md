## Limited Scope Audit Quick Reference

### What Makes a Valid Certification?

| Requirement | Details |
|-------------|---------|
| Qualifying Institution | Bank, trust company, or insurance carrier |
| Regulated | By federal or state agency |
| Required Content | Fair values, income, gains/losses, transactions |
| Required Statements | Complete, accurate, prepared per GAAP |
| Signature | Authorized official |
| Coverage | Entire plan year |

### Audit Phases & Typical Timeline

| Phase | Duration | Key Activities |
|-------|----------|----------------|
| Pre-Engagement | 1-2 weeks | Engagement letter, PBC request |
| Planning | 3-4 weeks | Risk assessment, obtain certification |
| Fieldwork | 2-3 weeks | Testing contributions, distributions, participant data |
| Completion | 2-3 weeks | Draft financials, review, issue report |

### Common Limited Scope Testing Areas

- ✓ Contributions (deferrals, match, profit sharing)
- ✓ Distributions (terminations, hardship, RMDs)
- ✓ Participant data (eligibility, vesting, compensation)
- ✓ Loans (compliance with 72(p), defaults)
- ✓ Forfeitures (calculation, timing, use)
- ✓ Administrative expenses
- ✓ ERISA compliance (fidelity bond, notices, Form 5500)
- ✗ Investment existence (covered by certification)
- ✗ Investment valuation (covered by certification)
