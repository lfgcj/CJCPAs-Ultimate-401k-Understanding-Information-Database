## Quick Reference

### Key Numbers (2024)

| Item | Amount |
|------|--------|
| Employee Contribution Limit | $23,000 |
| Catch-Up Contribution (50+) | $7,500 |
| Total Annual Addition Limit | $69,000 |
| Compensation Limit | $345,000 |
| Highly Compensated Employee Threshold | $155,000 |
| Key Employee (Officer) Threshold | $220,000 |
| Audit Requirement Threshold | 100+ eligible participants |

### Important Deadlines

| Deadline | Description |
|----------|-------------|
| Form 5500 Filing | 7 months after plan year end (July 31 for calendar year) |
| Extension | Up to 2.5 months with Form 5558 |
| Participant Disclosures | Within 90 days of becoming participant |
| SAR Distribution | Within 9 months of plan year end |
| ADP/ACP Corrections | 2.5 months after plan year end for QNEC refunds |
