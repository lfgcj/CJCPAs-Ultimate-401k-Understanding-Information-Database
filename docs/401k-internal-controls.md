# 401(k) Internal Controls Testing

## Overview

Internal controls are the policies and procedures that ensure plan operations are conducted properly. During a 401(k) audit, the auditor evaluates internal controls to determine their effectiveness and to plan the audit approach.

## Internal Control Framework

### COSO Framework

Most plans use the COSO Internal Control Framework:
- Control Environment
- Risk Assessment
- Control Activities
- Information and Communication
- Monitoring Activities

### Purpose of Internal Controls

Controls help ensure:
- Accurate financial reporting
- Compliance with laws and regulations
- Effective operations
- Safeguarding of assets
- Prevention and detection of errors and fraud

### Control Objectives for 401(k) Plans

| Objective | Description |
|-----------|-------------|
| Existence | Assets and liabilities exist |
| Completeness | All transactions recorded |
| Rights | Plan has rights to assets |
| Valuation | Amounts properly valued |
| Presentation | Proper classification and disclosure |

## Risk Assessment

### Risk Identification

Common 401(k) plan risks:

| Area | Risks |
|------|-------|
| Contributions | Late deposits, calculation errors |
| Participant data | Inaccurate records, eligibility errors |
| Investments | Valuation errors, unauthorized transactions |
| Benefits | Incorrect distributions, vesting errors |
| Loans | Non-compliance, defaults |
| Service providers | Errors, fraud, security breaches |

### Risk Evaluation

Consider:
- Likelihood of occurrence
- Potential impact
- Existing controls
- Residual risk

### Risk Response

| Response | Description |
|----------|-------------|
| Accept | Risk within tolerance |
| Avoid | Eliminate activity causing risk |
| Reduce | Implement controls |
| Share | Transfer risk (insurance) |

## Control Activities

### Types of Controls

| Type | Description |
|------|-------------|
| Preventive | Stop errors before they occur |
| Detective | Identify errors after they occur |
| Corrective | Fix errors that are detected |
| Manual | Performed by people |
| Automated | Performed by systems |

### Key Control Activities

1. **Authorization**
   - Approval of transactions
   - Defined authority limits
   - Documentation requirements

2. **Segregation of Duties**
   - Separate initiation, approval, recording
   - No single person controls process
   - Independent reconciliation

3. **Verification**
   - Review of calculations
   - Confirmation of data
   - Comparison to expectations

4. **Reconciliation**
   - Account balances to records
   - Participant totals to trust
   - Service provider reports

5. **Physical Controls**
   - Secure document storage
   - Access restrictions
   - Asset protection

## Control Deficiencies

### Classification of Deficiencies

| Classification | Definition |
|----------------|------------|
| Deficiency | Control doesn't operate effectively |
| Significant Deficiency | More than remote likelihood of material misstatement |
| Material Weakness | Reasonable possibility of material misstatement |

### Evaluation Factors

Consider:
- Nature of financial statement accounts
- Susceptibility to loss or fraud
- Complexity of processing
- Volume of transactions
- Importance of activity

### Common Control Deficiencies

| Deficiency | Description |
|------------|-------------|
| Inadequate segregation | Same person initiates and approves |
| Missing approvals | Transactions not authorized |
| No reconciliation | Accounts not reconciled |
| Untimely review | Reviews performed too late |
| Inadequate documentation | No evidence of control operation |
| System access | Inappropriate access permissions |

### Remediation

Steps to address deficiencies:
1. Document the deficiency
2. Determine root cause
3. Design corrective action
4. Implement changes
5. Test effectiveness
6. Monitor ongoing

## SOC 1 Reports

### What is a SOC 1 Report?

Service Organization Control report:
- Covers controls at service organizations
- Relevant to user entity financial reporting
- Prepared by independent auditor

### Types of SOC 1 Reports

| Type | Coverage |
|------|----------|
| Type I | Design of controls at point in time |
| Type II | Design and operating effectiveness over period |

### Reading a SOC 1 Report

Key sections:
1. **Management's Description** - System and controls described
2. **Service Auditor's Opinion** - Auditor's conclusions
3. **Control Objectives** - What controls designed to achieve
4. **Tests of Controls** - Procedures performed
5. **Results of Tests** - Exceptions identified
6. **Complementary User Entity Controls** - User responsibilities

### Complementary User Entity Controls (CUECs)

Controls the plan must perform:
- Review service provider reports
- Reconcile data to provider records
- Authorize transactions
- Monitor service levels
- Report issues promptly

### Using SOC 1 Reports

| Action | Purpose |
|--------|---------|
| Obtain reports | Request from all service providers |
| Review coverage | Ensure period covers plan year |
| Evaluate exceptions | Assess impact of exceptions |
| Consider CUECs | Verify user controls in place |
| Bridge period | Consider controls after report period |

### When SOC 1 Not Available

Options:
- Request SOC 1 from provider
- Perform alternative procedures
- Obtain written representations
- Test controls directly

## Testing Controls for Audit

### Understanding Controls

Auditor procedures:
- Inquiry of personnel
- Observation of operations
- Inspection of documentation
- Walkthrough of transactions

### Testing Design

Evaluate whether control:
- Addresses identified risks
- Would prevent/detect misstatements
- Is designed appropriately

### Testing Operating Effectiveness

| Test Type | Description |
|-----------|-------------|
| Inquiry | Ask how control operates |
| Observation | Watch control performed |
| Inspection | Review evidence of control |
| Reperformance | Redo the control procedure |

### Sample Sizes for Control Testing

Depends on:
- Frequency of control
- Reliance level
- History of exceptions
- Other controls in place

| Frequency | Typical Sample |
|-----------|---------------|
| Daily | 25-40 |
| Weekly | 15-25 |
| Monthly | 5-12 |
| Quarterly | 4 |
| Annually | 1 |
