## Disclosure Requirements Overview

### Primary Disclosure Requirements

| Disclosure | Authority | Timing |
|------------|-----------|--------|
| SPD | ERISA Section 102 | 90 days after eligibility |
| SAR | ERISA Section 104 | 9 months after plan year end |
| Fee Disclosure (404a-5) | DOL Regulation | Initial and annual |
| Investment Information | 404(c) | Quarterly |
| Benefit Statements | ERISA Section 105 | Quarterly (participant-directed) |
| Safe Harbor Notice | IRC Section 401(k)(12) | 30-90 days before plan year |
| QDIA Notice | DOL Regulation 2550.404c-5 | 30+ days before |

### Delivery Methods

| Method | Requirements |
|--------|--------------|
| First-class mail | Standard delivery |
| Hand delivery | In-person at work |
| Electronic | If meet DOL electronic delivery rules |
| Website | For certain disclosures with notice |

### Electronic Delivery Requirements

To deliver electronically:
- Employee must have access to electronic medium at work, OR
- Employee must affirmatively consent
- System must ensure actual receipt
- Participant can request paper copy

## Summary Plan Description (SPD)

### Purpose

Plain-language document explaining plan features and participant rights.

### Required Contents

| Element | Description |
|---------|-------------|
| Plan name | Official name of plan |
| Plan sponsor | Employer information |
| Plan administrator | Contact information |
| Agent for service | Legal process recipient |
| Plan type | 401(k), profit sharing, etc. |
| Plan number | 3-digit number |
| Plan year | Beginning and ending dates |
| Eligibility | Requirements to participate |
| Contributions | Types and formulas |
| Vesting | Schedule and calculation |
| Distributions | When and how available |
| Claims procedures | How to file and appeal |
| ERISA rights | Statement of participant rights |

### Distribution Requirements

| Event | Deadline |
|-------|----------|
| Initial enrollment | Within 90 days of eligibility |
| Material modifications | Within 210 days of plan year end |
| Participant request | Within 30 days |
| Updated SPD | Every 5 years (if changed) or 10 years |

### Summary of Material Modifications (SMM)

When plan is amended:
- Provide within 210 days of plan year end
- Describe changes made
- Plain language requirement
- May be incorporated into updated SPD

## Summary Annual Report (SAR)

### Purpose

Annual summary of plan financial information derived from Form 5500.

### Required Contents

| Element | Description |
|---------|-------------|
| Basic financial information | Assets, expenses |
| Rights to additional information | How to request |
| Contact information | Plan administrator |
| Statement of plan status | Basic plan information |

### Distribution Timeline

| Filing Type | SAR Due |
|-------------|---------|
| Regular filing | 9 months after plan year end |
| Extended filing | 2 months after Form 5500 filed |

### Sample SAR Language

Required statements include:
- Total plan assets
- Expenses paid
- Right to receive Form 5500
- ERISA rights statement

## Participant Fee Disclosures

### 404a-5 Disclosure Requirements

DOL regulation requiring fee disclosure to participants.

### Initial Disclosure

Provided when participant first eligible:
- General plan administrative information
- Individual expenses
- Investment options available
- Fee and expense information

### Annual Disclosure

Provided annually:
- Updated fee information
- Any changes from prior disclosure
- Investment performance
- Fee comparisons

### Quarterly Disclosure

Included with benefit statements:
- Dollar amount of fees charged
- Description of services
- Investment expenses

### Required Investment Information

| Category | Information Required |
|----------|---------------------|
| Name and type | Fund name, asset class |
| Performance | 1, 5, 10-year returns |
| Benchmark | Comparison benchmark |
| Fees | Expense ratios, other fees |
| Website | Where to get more information |

### Comparative Chart Format

```
| Investment    | Type        | 1 Year | 5 Year | 10 Year | Expense |
|---------------|-------------|--------|--------|---------|---------|
| Growth Fund   | Large Cap   | 12.5%  | 10.2%  | 8.7%    | 0.45%   |
| Value Fund    | Large Cap   | 8.3%   | 9.1%   | 7.8%    | 0.55%   |
| Bond Fund     | Fixed Income| 3.2%   | 4.1%   | 4.5%    | 0.25%   |
| Target 2040   | Asset Alloc | 9.8%   | 8.5%   | 7.2%    | 0.65%   |
```

## Investment Disclosures

### ERISA Section 404(c)

If plan wants protection from investment decisions:
- Provide sufficient investment information
- Allow reasonable transfer frequency
- Offer diversified options

### Required Information

| Information | Description |
|-------------|-------------|
| Investment objectives | Goals of each option |
| Risk characteristics | Volatility, risk level |
| Performance history | Returns over time |
| Fees and expenses | All costs to participant |
| Portfolio composition | What fund holds |
| Voting rights | For employer securities |

### Frequency

| Disclosure | Timing |
|------------|--------|
| On enrollment | Complete information |
| Quarterly | Statements with fees |
| Annually | Updated disclosure |
| On request | Within 30 days |

## Safe Harbor Notices

### Purpose

Explain safe harbor features and participant rights.

### Timing

| Timing | Description |
|--------|-------------|
| Initial | Before participant eligible |
| Annual | 30-90 days before plan year |

### Required Contents

| Element | Description |
|---------|-------------|
| Safe harbor formula | Contribution type and amount |
| Vesting | 100% immediate (or QACA 2-year) |
| Withdrawal restrictions | Distribution limitations |
| Type of contribution | Match or non-elective |
| How to make deferral elections | Enrollment instructions |

### Sample Safe Harbor Notice Language

```
SAFE HARBOR MATCHING CONTRIBUTION NOTICE

The Plan provides a Safe Harbor Matching Contribution as follows:

- 100% of your elective deferrals up to 3% of compensation, PLUS
- 50% of your elective deferrals from 3% to 5% of compensation

Maximum employer match: 4% of compensation

This contribution is 100% vested immediately.

To receive the full match, you should defer at least 5% of 
your compensation.
```

## QDIA Notices

### Purpose

Inform participants about default investment if no election made.

### When Required

- Automatic enrollment plans
- Any plan with default investment
- When contributions default to QDIA

### Timing

| Timing | Description |
|--------|-------------|
| Initial | 30+ days before first investment |
| Annual | 30+ days before each plan year |

### Required Contents

| Element | Description |
|---------|-------------|
| QDIA description | What the default investment is |
| Right to opt out | How to make different choice |
| Investment information | Performance, fees |
| Right to transfer | How to move out of QDIA |

## Automatic Enrollment Notices

### EACA Notice

For Eligible Automatic Contribution Arrangements:
- Default deferral percentage
- Right to elect out
- How to change election
- Investment default

### QACA Notice

For Qualified Automatic Contribution Arrangements:
- All EACA requirements plus
- Auto-escalation feature
- Safe harbor contribution
- Can combine with Safe Harbor notice

### Timing

30-90 days before:
- Plan year begins
- Participant becomes eligible
- Automatic features apply

## Distribution Notices

### 402(f) Notice

Required before eligible rollover distribution:
- Tax consequences
- Rollover options
- Direct vs indirect rollover
- Withholding rules

### Timing

30-180 days before distribution.

### Required Topics

| Topic | Description |
|-------|-------------|
| Direct rollover | Option to roll over directly |
| Mandatory withholding | 20% if not direct rollover |
| Tax treatment | How distribution is taxed |
| Penalties | 10% early withdrawal if applicable |
| Special rules | NUA, periodic payments |

### QJSA/QPSA Notice

For plans subject to survivor annuity rules:
- Explain joint and survivor annuity
- Waiver rights
- Spousal consent requirements
- Financial effect of waiver

## Other Required Notices

### Blackout Notice

Required when participant investment rights suspended:
- 30+ days before blackout
- Explain what rights affected
- Duration of blackout
- Reason for blackout

### Benefit Statement

| Plan Type | Frequency |
|-----------|-----------|
| Participant-directed | Quarterly |
| Not participant-directed | Annually |

### PBGC Notice

Generally not applicable to 401(k) plans (defined contribution plans not covered by PBGC).

### Domestic Relations Orders

When QDRO received:
- Notify participant and alternate payee
- Explain determination procedures
- Provide determination timeline

### Plan Amendment Notice

204(h) notice required for:
- Significant reduction in future benefit accrual
- 15 days before effective date (45 for large plans)

## Communication Best Practices

### Plain Language

| Do | Don't |
|----|-------|
| Use simple words | Use jargon |
| Short sentences | Long paragraphs |
| Active voice | Passive voice |
| Examples | Abstract concepts only |

### Effective Communications

| Practice | Benefit |
|----------|---------|
| Multiple channels | Reach all participants |
| Regular timing | Predictable communication |
| Clear subject lines | Easy to identify |
| Action items clear | Know what to do |
| Contact information | Know who to ask |

### Documenting Delivery

Maintain records of:
- What was sent
- When it was sent
- To whom
- Method of delivery
- Proof of delivery (if available)
