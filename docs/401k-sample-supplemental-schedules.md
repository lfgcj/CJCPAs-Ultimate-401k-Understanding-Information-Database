## Overview of Required Schedules

### When Schedules Are Required

| Schedule | Required When |
|----------|--------------|
| Schedule of Assets (Held at End of Year) | Always required for large plans |
| Schedule of Assets (Acquired and Disposed) | Assets both acquired AND disposed in same year |
| Schedule of Delinquent Participant Contributions | Late deposits occurred during year |
| Schedule G Part I - Loans/Fixed Income | Loans or fixed income in default/uncollectible |
| Schedule G Part II - Leases in Default | Leases in default (rare for 401(k)) |
| Schedule G Part III - Nonexempt Transactions | Prohibited transactions not corrected |

### Auditor's Opinion on Schedules

The auditor provides an opinion on the supplemental schedules in relation to the financial statements as a whole. This opinion is included in the limited scope audit report.

## Schedule H Line 4i - Schedule of Assets (Held at End of Year)

### Requirements

Per DOL regulations, this schedule must include:
- Identity of issue, borrower, lessor, or similar party
- Description of investment (including maturity date, rate of interest, collateral, par or maturity value)
- Current value
- Cost (for nonparticipant-directed investments only)
- Indication of party-in-interest transactions

### Detailed Column Requirements

#### Column (a) - Party-in-Interest

Mark with asterisk (*) if the investment involves:
- Plan sponsor or affiliates
- Plan fiduciaries
- Service providers to the plan
- Employees of the plan sponsor
- Relatives of the above
- Entities owned 50%+ by parties above

**Common party-in-interest items:**
- Participant loans (always)
- Employer securities (always)
- Investments managed by the plan trustee
- Service provider proprietary funds

#### Column (b) - Identity of Issue

| Investment Type | What to Include |
|-----------------|-----------------|
| Mutual funds | Fund company name |
| Collective trusts | Trustee name |
| Stocks | Company name |
| Bonds | Issuer name |
| Participant loans | "Participant loans" |
| Insurance contracts | Insurance company name |

#### Column (c) - Description

| Investment Type | Required Description |
|-----------------|---------------------|
| Mutual funds | Fund name |
| Stocks | Type of stock, par value, number of shares |
| Bonds | Interest rate, maturity date, par value |
| Participant loans | Interest rate range, maturity range |
| Real estate | Property description, location |
| Insurance contracts | Contract type, contract number |

#### Column (d) - Cost

- **Required** for non-participant directed investments
- Use "**" notation for participant-directed investments
- If cost not determinable, use fair value at acquisition

#### Column (e) - Current Value

- Fair value as of plan year-end
- Must agree to financial statement investments
- Plus participant loans
- Totals must reconcile

## Schedule H Line 4j - Schedule of Assets (Acquired and Disposed Within Year)

### When Required

Only required when assets were **both** acquired **and** disposed of within the same plan year. Not commonly required for typical 401(k) plans with mutual fund investments.

## Schedule H Line 4a - Schedule of Delinquent Participant Contributions

### When Required

Required when participant contributions (including deferrals and loan repayments) were not transmitted to the trust within the time period prescribed by DOL regulations.

### DOL Timing Requirements

| Plan Size | Deposit Requirement |
|-----------|---------------------|
| All plans | As soon as reasonably possible, but no later than 15th business day of following month |
| Small plans (<100) | 7 business days is safe harbor |
| Large plans (100+) | Typically 1-3 business days based on facts and circumstances |

### Lost Earnings Calculation

The DOL provides an online calculator for determining lost earnings:
https://www.dol.gov/agencies/ebsa/about-ebsa/our-activities/enforcement/vfcp-calculator

```
LOST EARNINGS CALCULATION EXAMPLE

Contribution Amount:           $12,345
Date Contribution Withheld:    01/10/2024
Date Deposited to Trust:       01/22/2024
Days Late:                     12 calendar days (8 business days)

DOL VFCP Calculator Inputs:
- Principal Amount: $12,345
- Loss Date: 01/10/2024
- Recovery Date: 01/22/2024
- Final Payment Date: 01/22/2024 (same as recovery)

Lost Earnings:                 $234 (based on IRC underpayment rate)
```

## Preparation Guidelines

### Schedule of Assets Preparation Checklist

| Step | Procedure | Completed |
|------|-----------|-----------|
| 1 | Obtain year-end investment listing from custodian | ☐ |
| 2 | Agree total investments to financial statements | ☐ |
| 3 | Add participant loans to schedule | ☐ |
| 4 | Verify party-in-interest identification | ☐ |
| 5 | Verify investment descriptions are complete | ☐ |
| 6 | Verify cost column is correct (or **) | ☐ |
| 7 | Verify all required columns are completed | ☐ |
| 8 | Foot and crossfoot schedule | ☐ |
| 9 | Agree totals to Form 5500 Schedule H | ☐ |

### Delinquent Contribution Schedule Preparation

| Step | Procedure | Completed |
|------|-----------|-----------|
| 1 | Identify all late deposits during year | ☐ |
| 2 | Calculate lost earnings for each | ☐ |
| 3 | Document deposits of late amounts | ☐ |
| 4 | Document deposits of lost earnings | ☐ |
| 5 | Classify as corrected vs. uncorrected | ☐ |
| 6 | Verify totals agree to Form 5500 | ☐ |
| 7 | Document VFCP filing (if applicable) | ☐ |

## Common Errors to Avoid

### Schedule of Assets Errors

| Error | How to Avoid |
|-------|--------------|
| Missing participant loans | Always include loans on schedule |
| Incorrect party-in-interest marking | Review service provider list |
| Missing investment descriptions | Include all required elements |
| Totals don't agree to F/S | Reconcile before finalizing |
| Cost shown for participant-directed | Use ** notation instead |
| Missing stable value/collective trusts | Include all investment types |

### Delinquent Contribution Errors

| Error | How to Avoid |
|-------|--------------|
| Not identifying all late deposits | Review all payroll periods |
| Incorrect lost earnings calculation | Use DOL calculator |
| Not reporting corrected amounts | Include all late deposits |
| Missing VFCP documentation | Maintain complete records |
| Inconsistent with Form 5500 | Reconcile before filing |

### Format and Presentation Errors

| Error | How to Avoid |
|-------|--------------|
| Missing EIN and plan number | Include on all schedules |
| Missing column headings | Use standard DOL format |
| Missing auditor's report reference | Include standard language |
| Incorrect date references | Use plan year-end date |
| Missing page numbers | Number all pages |
