# 401(k) Plan Documents

## Overview

The plan document is the legal foundation of a 401(k) plan. It defines how the plan operates and must comply with applicable laws and regulations. Understanding plan document requirements is essential for proper plan administration.

## Plan Document Requirements

### Legal Requirement

ERISA Section 402 requires:
- Every employee benefit plan be established and maintained pursuant to a written document
- Plan must contain required provisions
- Plan must comply with ERISA and IRC

### Purpose of Plan Document

| Purpose | Description |
|---------|-------------|
| Legal compliance | Satisfy IRS and DOL requirements |
| Operational guide | How to administer the plan |
| Participant rights | What participants are entitled to |
| Fiduciary guidance | Rules for fiduciary conduct |
| Dispute resolution | Resolve questions and disputes |

### Key Document Sections

| Section | Content |
|---------|---------|
| Definitions | Key terms defined |
| Eligibility | Who can participate |
| Contributions | Types and formulas |
| Vesting | Ownership of benefits |
| Investments | Investment options and procedures |
| Distributions | When and how benefits paid |
| Administration | How plan is managed |
| Amendment/Termination | How plan can be changed or ended |

## Types of Plan Documents

### Prototype Plans

**Description:**
- Standardized documents sponsored by financial institutions
- Pre-approved by IRS
- Limited customization options

**Advantages:**
- Lower cost
- IRS pre-approved
- Regular updates by sponsor
- Simpler to maintain

**Disadvantages:**
- Less flexibility
- May include unwanted provisions
- Limited customization

### Volume Submitter Plans

**Description:**
- Semi-customized template
- Sponsored by practitioners/TPAs
- Pre-approved by IRS with modifications allowed

**Advantages:**
- More flexibility than prototype
- IRS approval of basic document
- Can accommodate many plan designs

**Disadvantages:**
- May require IRS determination for modifications
- More complex than prototype

### Individually Designed Plans

**Description:**
- Custom-drafted for specific employer
- Maximum flexibility
- Requires own IRS determination letter

**Advantages:**
- Complete flexibility
- Tailored to employer needs
- Any permissible design

**Disadvantages:**
- Higher cost
- Must obtain own determination letter
- More maintenance responsibility

### Comparison

| Feature | Prototype | Volume Submitter | Individual |
|---------|-----------|------------------|------------|
| Flexibility | Low | Medium | High |
| Cost | Low | Medium | High |
| IRS approval | Automatic | Automatic/Individual | Individual |
| Maintenance | Provider | Provider/Employer | Employer |
| Customization | Limited | Moderate | Complete |

## Required Provisions

### ERISA Requirements

ERISA Section 402(b) requires:
- Named fiduciaries
- Procedure for establishing and carrying out funding policy
- Amendment procedure
- Basis for allocations and benefits

### IRC Qualification Requirements

IRC Section 401(a) requires provisions for:

| Requirement | IRC Section |
|-------------|-------------|
| Exclusive benefit | 401(a)(2) |
| Participation | 401(a)(3), 410 |
| Vesting | 401(a)(7), 411 |
| Joint and survivor annuity | 401(a)(11) |
| Limitations on benefits | 401(a)(16), 415 |
| Top-heavy rules | 401(a)(10), 416 |
| Nondiscrimination | 401(a)(4) |
| Distribution rules | 401(a)(9) |
| Elective deferrals | 401(k) |

### Must-Have Provisions

| Provision | Requirement |
|-----------|-------------|
| Plan name | Official name |
| Plan sponsor | Employer information |
| Plan administrator | Who administers |
| Entry dates | When employees enter |
| Eligibility | Requirements to participate |
| Compensation definition | Pay for contribution purposes |
| Contribution formulas | How contributions calculated |
| Vesting schedule | When contributions owned |
| Distribution events | When benefits paid |
| Amendment procedure | How plan changed |
| Termination procedure | How plan ends |

## Plan Amendments

### When Amendments Required

| Event | Amendment Deadline |
|-------|-------------------|
| Law change | Per IRS guidance |
| Plan design change | Before effective date |
| Discretionary change | Before effective date |
| Correction | Per EPCRS |
| Document update | Per remedial amendment period |

### Amendment Process

1. **Identify need** - Law change or desired change
2. **Draft amendment** - Prepare language
3. **Review for compliance** - Legal review
4. **Authorize adoption** - Proper authority
5. **Execute amendment** - Signatures
6. **Communicate** - Notify participants if required
7. **Implement** - Operate per amendment

### Retroactive Amendments

Generally:
- Remedial amendments can be retroactive
- Must not reduce accrued benefits
- Must be within remedial amendment period
- Favorable treatment may require timeliness

### Anti-Cutback Rule

IRC Section 411(d)(6) prohibits:
- Reducing accrued benefits
- Eliminating optional forms
- Eliminating protected benefits

Exceptions:
- Permitted by regulation
- Required by law
- Participant election

## Adoption Agreements

### Purpose

Adoption agreement allows employer to:
- Select options within pre-approved document
- Customize within approved parameters
- Maintain separate elections

### Contents

| Section | Purpose |
|---------|---------|
| Elections | Employer choices |
| Effective dates | When provisions apply |
| Plan information | Plan-specific data |
| Signature page | Authorized adoption |

### Relationship to Basic Plan Document

```
Basic Plan Document (Legal language)
           +
Adoption Agreement (Employer elections)
           =
Complete Plan Document
```

## Trust Agreements

### Purpose

Trust agreement:
- Establishes plan trust
- Appoints trustee(s)
- Defines trustee duties
- Governs asset handling

### Types of Trustees

| Type | Responsibilities |
|------|-----------------|

| Discretionary Trustee | Makes investment decisions |
| Corporate Trustee | Bank or trust company |
| Individual Trustee | Named person(s) |

|-----------|---------|

| Trustee powers | Authority granted |
| Investment duties | Investment responsibility |
| Distributions | How to make distributions |

## IRS Determination Letters

### Purpose

IRS determination letter:
- Confirms plan qualified status
- Provides reliance for tax treatment
- Demonstrates compliance

### When Required

| Situation | Letter Needed |
|-----------|---------------|
| Prototype/Volume Submitter | Opinion/Advisory letter covers |
| Individually Designed | Need own determination |
| Plan termination | Final determination |
| Major changes | May request letter |

### Application Process

For individually designed plans:
1. Complete Form 5300
2. Gather required attachments
3. Pay user fee
4. Submit to IRS
5. Respond to any questions
6. Receive determination letter

### Reliance on Letter

| Type | Reliance |
|------|----------|
| Current letter | Qualified as of date |
| Opinion letter (prototype) | Reliance through adoption |
| Advisory letter (VS) | Reliance with limits |
| No letter | Plan may still be qualified |

### Remedial Amendment Period

Time to correct document defects:
- Extends from date of defect
- Through remedial amendment period
- Allows retroactive correction
- Per IRS guidance
