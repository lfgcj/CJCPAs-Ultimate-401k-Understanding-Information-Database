# 401(k) Plan Administration

## Overview

Effective 401(k) plan administration is the foundation of a successful retirement program. Plan sponsors and administrators bear significant responsibilities for ensuring the plan operates in compliance with all applicable laws while serving participants' best interests. This comprehensive guide covers all aspects of 401(k) plan administration.

## Plan Sponsor Responsibilities

### Core Duties

The plan sponsor, typically the employer, has ultimate responsibility for the plan:

| Responsibility | Description |
|---------------|-------------|
| Plan Establishment | Adopt plan document and establish trust |
| Fiduciary Oversight | Ensure proper fiduciary governance |
| Funding | Remit contributions timely |
| Compliance | Maintain qualified plan status |
| Provider Oversight | Select and monitor service providers |
| Reporting | File required government reports |

### Settlor vs Fiduciary Functions

Understanding the distinction is critical for liability purposes:

#### Settlor Functions (Non-Fiduciary)

Business decisions about plan design:
- Establishing or terminating the plan
- Designing plan features
- Amending plan provisions
- Deciding to merge plans
- Setting contribution formulas

#### Fiduciary Functions

Actions subject to ERISA fiduciary standards:
- Investment selection and monitoring
- Participant benefit determinations
- Plan interpretation
- Service provider selection
- Expense allocation decisions

### Decision-Making Framework

```
Plan Design Decision → Settlor Function → Business Judgment Standard
         ↓
Plan Operation Decision → Fiduciary Function → Prudent Expert Standard
```

### Documentation Requirements

Plan sponsors must maintain proper documentation:

| Document | Purpose | Retention |
|----------|---------|-----------|
| Board/Committee resolutions | Authorize plan actions | Permanent |
| Meeting minutes | Document decision-making | 6+ years |
| Plan amendments | Record plan changes | Permanent |
| Service agreements | Define provider relationships | Life of contract + 6 years |
| Participant communications | Prove compliance | 6+ years |

## Plan Administrator Role

### Definition

The Plan Administrator is identified in the plan document and has primary responsibility for:
- Day-to-day plan operations
- Filing Form 5500
- Providing required disclosures
- Responding to participant inquiries
- Processing claims

### Who Can Serve

| Option | Pros | Cons |
|--------|------|------|
| Employer | Direct control | Resource demands |
| Committee | Shared responsibility | Coordination needed |
| Third-Party Administrator | Expertise | Cost |
| Named individual | Accountability | Single point of failure |

### Fiduciary Status

The Plan Administrator is a named fiduciary under ERISA:
- Must act prudently
- Must act in participants' best interests
- Subject to co-fiduciary liability
- Personal liability for breaches

### Administrative Authority

| Authority | Description |
|-----------|-------------|
| Plan interpretation | Resolve ambiguities |
| Eligibility determinations | Apply plan terms |
| Claims decisions | Process benefit claims |
| Disclosure compliance | Distribute required notices |
| Recordkeeping | Maintain plan records |

### Delegation

The Plan Administrator may delegate duties:
- Must be prudent delegation
- Must monitor delegates
- Cannot delegate all responsibility
- Remains ultimately responsible
- Document all delegations

## Day-to-Day Administration

### Enrollment Administration

#### New Employee Enrollment

| Step | Timeline | Responsibility |
|------|----------|----------------|
| Identify eligibility | Before entry date | HR/Administrator |
| Provide SPD | Within 90 days of eligibility | Administrator |
| Provide enrollment materials | Before entry date | Administrator |
| Process elections | By entry date | Recordkeeper |
| Verify beneficiary designations | At enrollment | Administrator |

#### Automatic Enrollment

For plans with automatic enrollment:

| Component | Requirement |
|-----------|-------------|
| Default deferral rate | As specified in plan (e.g., 3%) |
| Auto-escalation | Annual increases (if applicable) |
| QDIA | Qualified default investment |
| Notice | 30+ days before each plan year |
| Opt-out | Allow participant choice |

### Election Changes

| Election Type | Processing Time | Frequency Allowed |
|---------------|-----------------|-------------------|
| Deferral percentage | Next payroll | Per plan (often unlimited) |
| Investment allocation | Same/next business day | Per plan |
| Roth vs pre-tax | Next payroll | Per plan |
| Beneficiary | Within reasonable time | Anytime |

### Termination Processing

When employees terminate:

| Step | Timeline |
|------|----------|
| Update employment status | Within 30 days |
| Calculate vested balance | At termination |
| Provide distribution options | When requested |
| Process forfeitures | Per plan terms |
| Remove from active testing | Year of termination |

## Contribution Administration

### Contribution Types

| Type | Timing | Responsibility |
|------|--------|----------------|
| Employee deferrals | Each payroll | Employer withholds |
| Employer match | Per plan terms | Employer contributes |
| Safe harbor | Per plan terms | Employer contributes |
| Profit sharing | End of year (typically) | Employer decides |
| QNEC/QMAC | As needed for testing | Employer contributes |

### Timely Deposit Requirements

#### DOL Safe Harbor

Contributions deposited within 7 business days:
- Small plans (under 100 participants)
- Creates rebuttable presumption of compliance

#### Large Plans

Contributions must be deposited:
- As soon as administratively feasible
- Generally within 1-3 business days
- Same timeline used consistently

#### Maximum Deadline

All contributions must be deposited:
- No later than 15th business day of following month
- This is NOT the target - it's the absolute maximum

### Contribution Processing Workflow

```
Payroll Processed
       ↓
Calculate Deferrals → Verify Elections → Withhold from Pay
       ↓
Calculate Match → Apply Formula → Determine Amount
       ↓
Remit to Trust → Deposit to Custodian → Allocate to Accounts
       ↓
Invest per Elections → Apply Participant Directions → Confirm Posting
```

### Contribution Reconciliation

Monthly reconciliation should verify:

| Element | Reconciliation |
|---------|---------------|
| Payroll totals | Match to remittance |
| Participant counts | Agree to census |
| By contribution type | Deferrals, match, profit sharing |
| To trust deposits | Bank statement agreement |
| To participant accounts | Recordkeeper confirmation |

### Contribution Corrections

| Error Type | Correction Method |
|------------|-------------------|
| Late deposit | Lost earnings calculation |
| Missed deferral | Corrective contribution + earnings |
| Incorrect match | Corrective contribution or refund |
| Over contribution | Refund with earnings |
| Wrong participant | Transfer with earnings |

## Participant Services

### Required Disclosures

| Disclosure | Timing | Content |
|------------|--------|---------|
| SPD | Within 90 days of eligibility | Plan terms and rights |
| SMM | 210 days after plan year of change | Material modifications |
| SAR | 9 months after plan year end | Financial summary |
| Fee disclosure (404a-5) | Initial and annual | Fees and expenses |
| Benefit statement | Quarterly (participant-directed) | Account information |
| QDIA notice | 30+ days before plan year | Default investment info |

### Communication Best Practices

| Channel | Use |
|---------|-----|
| Written notices | Required disclosures |
| Email | Timely updates (with consent) |
| Website | Investment information |
| Mobile app | Account access |
| In-person meetings | Enrollment, education |
| Webinars | Ongoing education |

### Claims and Appeals

#### Claims Procedure Requirements

| Element | Requirement |
|---------|-------------|
| Written procedures | In SPD |
| Initial decision | Within 90 days |
| Extension notice | If needed, 90 more days |
| Denial notice | Specific reasons, plan provisions |
| Appeal rights | Full and fair review |

#### Appeals Process

| Step | Timeline |
|------|----------|
| File appeal | Within 60 days of denial |
| Review | Full review of denial |
| Decision | Within 60 days |
| Final decision | Written explanation |

### Participant Data Management

| Data Element | Source | Update Frequency |
|--------------|--------|------------------|
| Name | HR records | As needed |
| Address | Participant | As needed |
| Date of birth | HR records | Initial enrollment |
| Date of hire | HR records | Initial enrollment |
| Compensation | Payroll | Each pay period |
| Hours | Payroll/Time system | Each pay period |
| Beneficiary | Participant | As designated |

## Compliance Monitoring

### Annual Compliance Calendar

#### Calendar Year Plan

| Month | Compliance Activity |
|-------|---------------------|
| January | Prior year close, begin testing |
| February | Finalize testing data |
| March | ADP/ACP correction deadline (QNEC) |
| March 15 | ADP/ACP refund deadline (no tax) |
| April | Top-heavy determination |
| July 31 | Form 5500 due (or extension) |
| September | SAR distribution deadline |
| October | Safe harbor notice |
| October 15 | Extended Form 5500 due |
| November | 404a-5 annual notice |
| December | Year-end contributions |

### Testing Requirements

| Test | Frequency | Deadline |
|------|-----------|----------|
| ADP/ACP | Annual | 2.5 months for QNEC |
| Top-Heavy | Annual | Prior year end |
| Coverage (410(b)) | Annual | Plan year end |
| 415 Limits | Per contribution | Ongoing |
| 402(g) Limit | Annual | Prior year amounts |

### Monitoring Checklist

```
Monthly:
☐ Contribution deposits timely
☐ Loan repayments current
☐ Participant data accurate
☐ Investment elections processed

Quarterly:
☐ Participant statements distributed
☐ Investment performance reviewed
☐ Fee review completed
☐ Compliance issues addressed

Annually:
☐ Nondiscrimination testing completed
☐ Form 5500 filed
☐ Required notices distributed
☐ Plan document reviewed
☐ Service provider review
☐ Fiduciary training completed
```

## Recordkeeping Requirements

### Document Retention

| Document Type | Retention Period |
|---------------|------------------|
| Plan document and amendments | Permanent |
| Trust agreement | Permanent |
| Form 5500 filings | 6 years |
| Participant records | 6 years after final distribution |
| Investment records | 6 years |
| Meeting minutes | 6 years |
| Contribution records | 6 years |
| Distribution records | 6 years after distribution |

### Required Records

| Category | Specific Records |
|----------|------------------|
| Plan establishment | Adoption agreement, trust, IRS determination |
| Participant data | Census, eligibility, elections |
| Contributions | Payroll, deposits, allocations |
| Investments | Statements, transactions, elections |
| Distributions | Requests, calculations, payments |
| Compliance | Testing results, corrections, notices |
| Governance | Minutes, resolutions, policies |

### Electronic Recordkeeping

Permissible if:
- Records are accurate and accessible
- Adequate backup and recovery
- Access controls in place
- Audit trail maintained
- Meets DOL electronic disclosure requirements

### Disaster Recovery

Essential elements:
- Offsite backup
- Recovery procedures
- Alternative access methods
- Testing of procedures
- Documentation of recovery plan

## Common Administrative Challenges

### Late Contributions

| Issue | Solution |
|-------|----------|
| Processing delays | Automate contribution processing |
| Payroll issues | Establish clear procedures |
| Cash flow problems | Never use participant funds |
| Discovery | Calculate lost earnings immediately |
| Correction | VFCP or self-correction |

### Data Errors

| Error Type | Prevention | Correction |
|------------|------------|------------|
| Eligibility | Clear criteria, automated tracking | Retroactive enrollment |
| Compensation | Proper definition, verification | Recalculation |
| Hours | Accurate time tracking | Service correction |
| Elections | Confirmation process | Process change |

### Compliance Failures

| Failure | Correction Program |
|---------|-------------------|
| Operational error | Self-Correction (SCP) |
| Plan document error | VCP |
| Audit discovered | Audit CAP |
| DOL error | VFCP |

### Participant Communication Issues

| Challenge | Solution |
|-----------|----------|
| Returned mail | Address verification procedures |
| Missing participants | Locate procedures |
| Language barriers | Translation services |
| Low engagement | Multiple communication channels |

### Technology Challenges

| Challenge | Solution |
|-----------|----------|
| System integration | API connections, testing |
| Data security | Encryption, access controls |
| Disaster recovery | Backup procedures |
| User adoption | Training, support |
