# 401(k) Audit Preparation Guide

## Overview

Proper preparation is essential for a smooth and efficient 401(k) audit. This guide provides a comprehensive checklist and timeline for preparing for your annual plan audit.

## Preparation Timeline

### 90 Days Before Year-End

| Task | Description |
|------|-------------|
| Review prior audit | Identify issues to address |
| Engage auditor | Confirm engagement or select new |
| Initial planning call | Discuss scope and timing |
| Identify coordinator | Designate audit contact |
| Begin document gathering | Start collecting permanent documents |

### Year-End Close

| Task | Description |
|------|-------------|
| Close participant accounts | Final year-end valuations |
| Reconcile trust | Trust to recordkeeper reconciliation |
| Reconcile participant data | Verify census accuracy |
| Calculate forfeitures | Process termination forfeitures |
| Review contribution timing | Identify any late deposits |

### 30 Days After Year-End

| Task | Description |
|------|-------------|
| Prepare trial balance | Financial statement starting point |
| Compile census data | Finalize participant information |
| Gather contribution records | All payroll periods |
| Compile distribution records | All payments made |
| Prepare loan schedule | Outstanding loans listing |

### Interim Fieldwork (60-90 Days)

| Task | Description |
|------|-------------|
| Provide documents | Submit all requested items |
| Answer questions | Respond to auditor inquiries |
| Address preliminary findings | Correct issues identified early |
| Provide SOC reports | Service provider reports |
| Confirm audit timeline | Verify target completion |

### Final Fieldwork (90-120 Days)

| Task | Description |
|------|-------------|
| Provide final data | Year-end statements and reports |
| Review draft reports | Financial statements and findings |
| Prepare management responses | Address observations |
| Sign representations | Management representation letter |
| Review final report | Before Form 5500 filing |

## Pre-Audit Internal Review

### Contribution Review

| Check | Action |
|-------|--------|
| Timing | Review deposit dates vs payroll dates |
| Accuracy | Recalculate sample of deferrals |
| Completeness | Verify all payrolls remitted |
| Matching | Verify match formula applied correctly |
| Limits | Check 402(g) and 415 compliance |

### Distribution Review

| Check | Action |
|-------|--------|
| Eligibility | Verify triggering events |
| Vesting | Recalculate sample vesting |
| Authorization | Confirm proper approvals |
| Tax reporting | Verify 1099-R accuracy |
| Timing | Verify timely processing |

### Participant Data Review

| Check | Action |
|-------|--------|
| Census accuracy | Compare to HR records |
| Eligibility | Verify entry dates correct |
| Status codes | Confirm active/terminated status |
| Compensation | Verify against payroll |
| Hours | Verify service tracking |

### Investment Review

| Check | Action |
|-------|--------|
| Reconciliation | Trust to recordkeeper |
| Valuation | Fair value verification |
| Transactions | Verify purchases/sales |
| Income | Verify dividends/interest |
| Fees | Verify expense allocations |

### Loan Review

| Check | Action |
|-------|--------|
| Compliance | Verify IRC 72(p) requirements |
| Repayments | Verify current repayment status |
| Defaults | Identify loans in default |
| Documentation | Verify loan agreements on file |

## Common Preparation Mistakes

### Document Issues

| Mistake | Solution |
|---------|----------|
| Missing documents | Create checklist and track |
| Outdated documents | Confirm current versions |
| Disorganized files | Organize by category |
| Multiple versions | Clearly label and date |
| Late submission | Submit well before deadlines |

### Reconciliation Issues

| Mistake | Solution |
|---------|----------|
| No reconciliation | Perform monthly reconciliations |
| Unexplained differences | Research and document |
| Timing differences | Identify and track |
| Unrecorded items | Review completeness |

### Process Issues

| Mistake | Solution |
|---------|----------|
| Single point of contact | Identify backup personnel |
| Delayed responses | Set response deadlines |
| Incomplete answers | Provide full information |
| Not addressing prior findings | Track and remediate |

## Working with Your Auditor

### Initial Planning Meeting

Discuss:
- Scope and timing
- Document requirements
- Key contacts
- Prior year issues
- Changes during year
- Known problems

### During Fieldwork

Best practices:
- Respond promptly to requests
- Provide complete information
- Ask clarifying questions
- Escalate issues early
- Keep communication open

### Draft Report Review

When reviewing:
- Verify accuracy of information
- Review financial statements carefully
- Understand any findings
- Prepare management responses
- Ask questions about anything unclear

### Communication Tips

| Do | Don't |
|----|-------|
| Be responsive | Delay responses |
| Be honest | Hide problems |
| Ask questions | Make assumptions |
| Document agreements | Rely on verbal |
| Escalate issues | Let problems fester |

## Post-Audit Follow-Up

### Address Findings

| Step | Action |
|------|--------|
| Review findings | Understand each issue |
| Develop action plan | How to correct |
| Implement corrections | Make necessary changes |
| Document | Keep records of corrections |
| Monitor | Ensure changes effective |

### Form 5500 Filing

| Task | Timeline |
|------|----------|
| Review final audit report | Before filing |
| Reconcile to Form 5500 | Ensure consistency |
| Attach required schedules | Include all attachments |
| File electronically | Via EFAST2 |
| Retain records | For required period |

### Lessons Learned

After each audit:
- Document what went well
- Identify areas for improvement
- Update procedures
- Train staff
- Plan for next year

### Ongoing Monitoring

Throughout the year:
- Monitor contribution timing
- Review participant data changes
- Track compliance issues
- Maintain documentation
- Stay current on regulations
