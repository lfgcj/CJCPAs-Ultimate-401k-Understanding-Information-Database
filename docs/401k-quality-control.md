## Quality Control Overview

### Professional Standards

| Standard | Focus |
|----------|-------|
| QC Section 10 | Firm's system of quality control |
| AU-C 220 | Quality control for an engagement |
| SAS 136 | Specific EBP audit quality requirements |

### Review Levels

| Level | Responsibility | Timing |
|-------|---------------|--------|
| Self-Review | Preparer | Continuous |
| Detailed Review | Manager/Senior Manager | During/after fieldwork |
| Partner Review | Engagement Partner | Before report issuance |
| EQR | Concurring Partner | High-risk engagements |

### Quality Control Objectives

- [ ] Work performed per professional standards
- [ ] Financial statements fairly presented per GAAP
- [ ] Sufficient appropriate audit evidence obtained
- [ ] Conclusions properly supported
- [ ] Report appropriate for circumstances
- [ ] Documentation complete and organized

## Common Review Notes

### Working Paper Issues

| Issue | Required Action |
|-------|-----------------|
| Missing date/initials | Add preparer date and initials |
| No conclusion documented | Add clear conclusion statement |
| Tickmark legend missing | Add tickmark explanations |
| Cross-reference incomplete | Complete cross-references |
| Source of data not identified | Document source of information |
| Sample not properly documented | Document population, selection method |
| Variance not explained | Investigate and document explanation |
| No tie-out to lead schedule | Complete tie-out |

### Substantive Testing Issues

| Issue | Required Action |
|-------|-----------------|
| Sample size not supported | Document basis for sample size |
| Selection method unclear | Document random/systematic selection |
| Exception not resolved | Investigate and resolve or project error |
| Testing incomplete | Complete all steps per program |
| Alternative procedures missing | Document why alternative used |

### Documentation Issues

| Issue | Required Action |
|-------|-----------------|
| Inquiry not documented | Prepare memo of inquiry |
| Client response not obtained | Follow up with client |
| Conclusion inconsistent with evidence | Revise conclusion or obtain additional evidence |
| Prior year issue not addressed | Document resolution or continued existence |

## Working Paper Documentation Standards

### General Requirements

Every working paper should include:

| Element | Requirement |
|---------|-------------|
| Heading | Plan name, plan year, section title |
| Preparer | Initials and date |
| Reviewer | Initials and date |
| Purpose | Clear statement of objective |
| Source | Where data was obtained |
| Procedure | What was done |
| Results | What was found |
| Conclusion | Assessment of results |

### Sample Documentation Template

```
┌─────────────────────────────────────────────────────────────────────┐
│ [PLAN NAME]                                                         │
│ Year Ended December 31, 20XX                                        │
│ [SECTION TITLE]                                                     │
│                                                                     │
│ Prepared by: _____ Date: _____    Reviewed by: _____ Date: _____   │
├─────────────────────────────────────────────────────────────────────┤
│ OBJECTIVE:                                                          │
│ [Clear statement of the purpose of this workpaper]                  │
├─────────────────────────────────────────────────────────────────────┤
│ SOURCE OF DATA:                                                     │
│ [Where the information was obtained - client, confirmation, etc.]   │
├─────────────────────────────────────────────────────────────────────┤
│ PROCEDURES PERFORMED:                                               │
│ 1. [First procedure]                                                │
│ 2. [Second procedure]                                               │
│ 3. [Third procedure]                                                │
├─────────────────────────────────────────────────────────────────────┤
│ RESULTS:                                                            │
│ [Detailed results of procedures]                                    │
│ [Include any exceptions noted]                                      │
├─────────────────────────────────────────────────────────────────────┤
│ CONCLUSION:                                                         │
│ [Clear statement of conclusion based on results]                    │
│ [If exceptions, note disposition]                                   │
│                                                                     │
│ Cross-Reference: [Reference to related workpapers]                  │
└─────────────────────────────────────────────────────────────────────┘
```
