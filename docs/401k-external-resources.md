## Annual Limits Quick Reference

### 2025 Plan Year Limits

| Limit | 2024 | 2025 | Change |
|-------|------|------|--------|
| **Elective Deferral (402(g))** | $23,000 | $23,500 | +$500 |
| **Catch-Up (50+)** | $7,500 | $7,500 | No change |
| **Enhanced Catch-Up (60-63)** | N/A | $11,250 | NEW |
| **Annual Addition (415)** | $69,000 | $70,000 | +$1,000 |
| **Compensation (401(a)(17))** | $345,000 | $350,000 | +$5,000 |
| **HCE Threshold** | $155,000 | $160,000 | +$5,000 |
| **Key Employee (Officer)** | $220,000 | $230,000 | +$10,000 |
| **SIMPLE Deferral** | $16,000 | $16,500 | +$500 |
| **SIMPLE Catch-Up** | $3,500 | $3,500 | No change |
| **Social Security Wage Base** | $168,600 | $176,100 | +$7,500 |

### Maximum Total Contributions (2025)

| Age Group | Deferral | Catch-Up | Total Employee | Max Annual Addition |
|-----------|----------|----------|----------------|---------------------|
| Under 50 | $23,500 | $0 | $23,500 | $70,000 |
| 50-59 | $23,500 | $7,500 | $31,000 | $70,000 |
| **60-63** | $23,500 | **$11,250** | **$34,750** | $70,000 |
| 64+ | $23,500 | $7,500 | $31,000 | $70,000 |

### Key Filing Deadlines (2025 Plan Year)

| Deadline | Description |
|----------|-------------|
| **March 15, 2026** | ADP/ACP corrective distributions (no excise tax) |
| **April 15, 2026** | Employer contribution deadline (calendar year, tax deadline) |
| **July 31, 2026** | Form 5500 due (calendar year plan) |
| **July 31, 2026** | Form 5558 extension request due |
| **September 30, 2026** | SAR distribution deadline |
| **October 15, 2026** | Extended Form 5500 deadline |
| **December 31, 2026** | QNEC/QMAC correction deadline |
